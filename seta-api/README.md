### Backend Setup

## Prerequisites
- Python (v3.10+)

1. **Create and activate virtual environment**:
   ```bash
   cd seta-api
   
   # On Windows
   python -m venv venv
   venv/Scripts/activate
   
   # On macOS/Linux
   python -m venv venv
   source venv/bin/activate

2. **Install dependencies**
    ```bash
    pip install -r requirements.txt

3. **Run The application**
    ```bash
    cd app
    python main.py
